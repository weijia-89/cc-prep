# Daily page user notes

This file is the persistent store for user notes on day pages (day-01.html through day-10.html). The cc-prep notes backend (scripts/notes_server.py) reads and writes this file as users add notes from the daily-page UI. The renderer (scripts/render_html.py) reads this file at build time and injects the matching block into each day page's left gutter.

Do not edit the delimiter comments directly. The notes backend manages them.
